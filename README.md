# Portfolio ARODE Drone

Site statique pret pour GitHub Pages.

## Publication rapide

1. Creer un depot GitHub, par exemple `portfolio`.
2. Envoyer le contenu de ce dossier a la racine du depot.
3. Dans GitHub, ouvrir `Settings > Pages`.
4. Choisir la branche `main` et le dossier `/root`.
5. Ouvrir l'URL GitHub Pages fournie par GitHub.

## Fichiers principaux

- `index.html` : page principale du portfolio.
- `assets/styles.css` : design responsive sombre oriente drone/PCB.
- `assets/app.js` : navigation mobile et animations legeres.
- `assets/docs/` : rapport, CV et documents techniques.
- `assets/media/` : images integrees au site.
