02 — Profil

Ce que je sais faire

Mon parcours en GEII m'a amene progressivement de l'electronique de base vers les systemes embarques complets : capteurs, cartes, programmation, robotique, logique numerique, interfaces et integration materielle.

En troisieme annee, le travail s'est oriente vers les drones intelligents et l'integration systeme : architecture X500, alimentation, communication, flux video, IA de detection et exploitation dans une interface operateur.

Le fil conducteur : comprendre comment un composant physique, un logiciel embarque et une chaine de traitement peuvent fonctionner ensemble de maniere fiable.
